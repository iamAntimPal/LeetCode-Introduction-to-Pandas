# 2881. Create a New Column

## Problem Statement

Write a solution to create a new column `bonus` that contains the doubled values of the `salary` column.

## DataFrame Schema

**DataFrame: employees**

| Column Name | Type   |
| ----------- | ------ |
| name        | object |
| salary      | int    |

## Example 1

### **Input:**

```rb
DataFrame employees
+---------+--------+
| name    | salary |
+---------+--------+
| Piper   | 4548   |
| Grace   | 28150  |
| Georgia | 1103   |
| Willow  | 6593   |
| Finn    | 74576  |
| Thomas  | 24433  |
+---------+--------+
```

### **Output:**

```rb
+---------+--------+--------+
| name    | salary | bonus  |
+---------+--------+--------+
| Piper   | 4548   | 9096   |
| Grace   | 28150  | 56300  |
| Georgia | 1103   | 2206   |
| Willow  | 6593   | 13186  |
| Finn    | 74576  | 149152 |
| Thomas  | 24433  | 48866  |
+---------+--------+--------+
```

### **Explanation:**
A new column `bonus` is created by doubling the value in the column `salary`.

## Constraints
- The `salary` column contains only positive integers.
- The function should return the modified DataFrame with the new `bonus` column.
