# 2884. Modify Columns

## Problem Statement

Write a solution to modify the `salary` column by multiplying each salary by 2.

## DataFrame Schema

**DataFrame: employees**

| Column Name | Type   |
| ----------- | ------ |
| name        | object |
| salary      | int    |

## Example 1

### **Input:**

```rb
+---------+--------+
| name    | salary |
+---------+--------+
| Jack    | 19666  |
| Piper   | 74754  |
| Mia     | 62509  |
| Ulysses | 54866  |
+---------+--------+
```

### **Output:**

```rb
+---------+--------+
| name    | salary |
+---------+--------+
| Jack    | 39332  |
| Piper   | 149508 |
| Mia     | 125018 |
| Ulysses | 109732 |
+---------+--------+
```

### **Explanation:**
Every salary has been doubled.

## Constraints
- The `salary` column contains integer values.
- The function should return the modified DataFrame with updated salary values.
